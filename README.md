### 1. 웹 사이트 소개

#### - 음식 레시피를 검색할 수 있는 웹 사이트(JavaScript)

- Recipe Search API(https://www.edamam.com/)를 통한 검색 데이터 제공
- 키워드 입력, 태그 버튼을 통한 레시피 검색 기능
- 레시피를 저장, 삭제할 수 있는 즐겨찾기 기능
